4 Displays basicos

Nesse modulo veremos os 4 tipos de DISPLAY basicos

Display em bloco ocupa toda a linha independente do tamnho do conteudo a tag = display: block; clicando em f12 na pagina vera q a visualizacao e em bloco ocupando toda area da pagina BLOCK FICAM SEMPRE UM EMBAIXO DO OUTRO

aplica-se o box model, margim e padding fucionam normalmente

Display inline ocupa somente ate onde vai o conteudo!! INLINE FICAM SEMPRE UM AO LADO DO OUTRO

box model nao se aplica o margin e padding funionam somente na horizontal

Display inline - block ocupa somente ate onde vai o conteudo!! INLINE-BLOCk FICAM SEMPRE UM AO LADO DO OUTRO

box model se aplica a margin e padding funcionam normalmente

Display-none oculta todo o display

o conteudo iserido no html fica oculto na visualizacao quando aplicamos o display none aplica-se quando voce quer ocultar alguma stilizacao da pagina de outros usuarios
