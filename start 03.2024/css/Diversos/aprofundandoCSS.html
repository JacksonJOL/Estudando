Especifidades no CSS

Prioridade dos seletores

seletores de menores prioridades, ele seleciona todos os elementos da pagina, porem qualquer outro seletor passa por cima dele

seletor univeral

utiliza para deixar uma caracteristica unica pra toda pagina.

tag

utilizando a tag como seletor todas as tags da pagina acatara a instrucao da tag EX: H4 color:red, todos os H4 da pagina ficariam vermelhos

id

cria-se um ID para determinar que aquele ID ira ter uma determinada configuracao especica H4 id="h4" color:"blue", somente esse H4 sera azul, os demais ficaram na cor padrao

css inline estilização na linha passa por cima de toda configuracao

TAG !important

é a prioridade mais alta passa por cima de toda e qualquer outra estilizacao
